# MP4_to_PPTX
将MP4文件输出为PPT

## 使用方法

1. 修改程序中的mp4文件的目录,修改输出图片的目标目录。设置合适的帧率：timeF。

![image-20231103230047856](./assets/image-20231103230047856.png)

2. 执行process.py 

3. 在目标文件夹中删除多余图片。

4. 修改out_ppt.py 中的目标图像目录，指定输出文件路径名称。

   ![image-20231103230317510](./assets/image-20231103230317510.png)

5. 执行out_ppt.py
